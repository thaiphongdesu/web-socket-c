# Simple WebSocket in Plain C

## ✨ Features (for now)

- 📖 **Read configuration file**  
  The library can parse a configuration file to load settings.

- 🌐 **Return simple HTML responses**  
  Basic HTTP responses are supported to serve static HTML content.

- 🔀 **Return simple HTML response with conditional tag**  
  The server can generate responses that include conditional tags based on logic.
